Software & Tools

The Department of Computer Science & Engineering integrates a wide range of software platforms, IDEs, and frameworks to provide students with practical exposure to both commercial and open-source technologies.

VS Code: Lightweight, customizable code editor with support for debugging and embedded Git control.

Eclipse: Popular IDE for Java, supports multiple languages.

Java SDK: Tools for Java application development.
