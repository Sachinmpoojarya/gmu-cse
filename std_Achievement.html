Real-Time Hand Sign Detection and Text Translation

• CSE Department

This project focuses on developing an AI-based system that detects hand signs (gestures) and translates them into text in real time. It aims to bridge the communication gap between sign language users (especially the hearing and speech-impaired community) and non-signers.

NeuroCalm Project – National Hackathon Achievement

Sachin, Abhiram, Prajwal, Sudharshan

Our students participated in the National Hackathon at Presidency University with their project NeuroCalm, an innovative solution for mental well-being. Competing against 80 teams nationwide, they showcased creativity and technical skills, securing the 10th position—a proud achievement for the Department of Computer Science & Engineering, GM University.

AI-Powered Investment Exit Strategy Predictor

Likitha C Y, Harshitha H M, Shreya B Yadav, Lakshmi B C

The AI-Powered Investment Exit Strategy Predictor helps investors make smarter, data-driven exit decisions by combining valuation forecasting, exit timing optimization, and sentiment analysis. Unlike static stop-loss rules, it adapts dynamically to market signals, offering clear recommendations with explanations to reduce emotional bias, protect gains, and maximize returns.

Code Quality Visualizer Project at Hackmitten 2.0

Sinchana M, Rohini KG, Pragathi S P, Prakruthi KN

At Hackmitten 2.0, a National Level Hackathon organized by Maharaja Institute of Technology, Thandavapura, our team developed Code Quality Visualizer—a tool for static analysis of code repositories. It highlights complexity hotspots, probable bugs, and includes an interactive dashboard with predictive insights to help developers address issues in advance.
